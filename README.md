# MusicEmotion
# 🎵 M.E (Music Emotion)

감정 기반 음악 추천 웹 서비스

---

## 📌 프로젝트 개요

**M.E (Music Emotion)** 는 사용자의 감정과 음악 장르를 기반으로 개인화된 음악을 추천하는 웹 서비스입니다.
"Music Emotion"의 약자이자, '나(Me)'라는 의미를 함께 담아 음악을 통해 감정을 치유하고 회복할 수 있도록 돕는 것을 목표로 합니다.

---

## 🧩 기획 배경 및 목적

대학 재학 중 4년간 밴드 동아리 활동을 하며 음악이 사람의 감정에 큰 영향을 미친다는 점을 체감했습니다. 기존 음악 추천 서비스는 청취 이력이나 선호 장르를 중심으로 추천이 이루어지지만, 사용자의 **현재 감정 상태**를 직접적으로 반영하지 못한다는 한계가 있습니다.

본 프로젝트는 사용자가 느끼는 감정에 맞는 음악을 추천함으로써 정서적 안정과 만족도를 높이고, 감정 기반의 새로운 음악 경험을 제공하는 것을 목적으로 합니다.

---

## 🎯 프로젝트 목표

실제 음악 스트리밍 서비스와 커뮤니티 플랫폼의 사용자 흐름을 분석하고, 감정 선택부터 음악 추천, 커뮤니티 공유까지의 과정을 하나의 서비스로 구현하는 것을 목표로 했습니다. 이를 통해 웹 서비스 전반의 구조를 이해하고, 백엔드 중심의 개발 역량을 강화하고자 했습니다.

---

## 🧪 벤치마킹

| 서비스     | 특징              | 벤치마킹 포인트     |
| ------- | --------------- | ------------ |
| 큐오넷     | 음악 제작 정보 및 커뮤니티 | 자유로운 소통 게시판  |
| inDJ    | 감정·상황 기반 음악 추천  | 감정 중심 개인화 추천 |
| Spotify | 대규모 음악 스트리밍 플랫폼 | 방대한 음악 라이브러리 |

**결과**: 감정 기반 추천(inDJ) + 커뮤니티 기능(큐오넷) + 음악 데이터 활용(Spotify)을 결합한 서비스 구조 설계

---

## 📋 주요 기능

### 👤 회원

* 로그인 / 회원가입
* 아이디 찾기, 비밀번호 변경 (이메일 인증)

### 🎶 음악

* 음악 검색
* 감정 + 장르 기반 음악 추천
* 곡 정보 조회

### 📂 마이페이지

* 플레이리스트 생성 / 수정 / 삭제
* 최근 재생 목록 및 통계 시각화
* 좋아요한 음악 관리
* 팔로우한 아티스트 / 앨범 조회

### 💬 커뮤니티

* 게시글 작성 / 수정 / 삭제
* 댓글 기능
* 공지 게시판 제공
