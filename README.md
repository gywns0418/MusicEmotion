# 🎵 M.E (Music Emotion)

감정 기반 음악 추천 웹 서비스 (졸업작품)

---

## 📌 프로젝트 한 줄 소개

사용자의 감정과 선호 장르를 기반으로 음악을 추천하고, 커뮤니티를 통해 음악 경험을 공유할 수 있는 웹 서비스입니다.

---

## 🎯 프로젝트 목표

실제 음악 서비스의 사용자 흐름을 분석하여 감정 선택 → 음악 추천 → 정보 조회 및 공유로 이어지는 기능을 구현함으로써, 웹 서비스 전반의 구조와 백엔드 개발 역량을 강화하는 것을 목표로 했습니다.

---

## 🧩 핵심 기능

* 감정 + 장르 기반 음악 추천
* 음악 검색 및 곡 정보 조회
* 플레이리스트 관리
* 최근 재생 목록 및 통계
* 좋아요 / 아티스트·앨범 팔로우
* 커뮤니티 게시판 및 댓글 기능

---

## 📸 화면 구성 및 주요 기능 설명

### 1. 감정 기반 음악 추천 화면

* 사용자가 현재 감정을 선택하고 선호 장르를 함께 선택
* 선택한 감정과 장르를 기준으로 음악 추천 결과 제공

![감정 기반 음악 추천 화면](./images/emotion_recommend.png)

---

### 2. 음악 검색 및 곡 정보 화면

* 키워드를 통한 음악 검색 기능 제공
* 곡 선택 시 아티스트, 앨범 등 상세 정보 확인 가능

![음악 검색 화면](./images/music_search.png)

---

### 3. 마이페이지 / 플레이리스트

* 플레이리스트 생성·수정·삭제 기능
* 최근 재생 목록과 통계 정보 시각화

![마이페이지 화면](./images/mypage.png)

---

### 4. 커뮤니티 기능

* 음악 추천 및 정보 공유를 위한 게시판 제공
* 게시글 및 댓글 작성 기능

![커뮤니티 화면](./images/community.png)
